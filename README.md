# AirPDF

Reemplazo de Adobe Acrobat Pro y Xodo. Standalone, instalable en Windows, IA opcional.

## Estado

En desarrollo (Fase 1: Lectura + Edicion).

## Build local

```powershell
pnpm install
pnpm tauri dev      # modo desarrollo
pnpm tauri build    # MSI release en src-tauri/target/release/bundle/msi/
```
